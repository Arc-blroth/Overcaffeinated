## Installation
  
- Manually:

  Download the [latest release](https://github.com/Arc-blroth/Overcaffinated/releases/latest) and install it manually using
  <kbd>Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from disk...</kbd>
